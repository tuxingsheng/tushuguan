# 快速开始

## 环境要求

- Node.js 18+
- Supabase 账号

## 安装

```bash
git clone <repo-url> && cd tushuguan
npm install
npm run dev        # → http://localhost:5173
npm run build      # 生产构建
```

## Supabase 初始化（首次必做）

1. [Supabase Dashboard](https://supabase.com/dashboard) 创建项目
2. **SQL Editor** → 粘贴执行 `supabase/schema.sql` 全部内容
3. **Authentication** → Providers → 启用 Email
4. **Authentication** → Users → Add User 创建管理员账号
5. 将 `URL` 和 `anon_key` 填入 `src/lib/supabase.js`

```js
// src/lib/supabase.js
import { createClient } from '@supabase/supabase-js'

const supabaseUrl = 'https://<your-project>.supabase.co'
const supabaseAnonKey = '<your-anon-key>'

export const supabase = createClient(supabaseUrl, supabaseAnonKey)
```

> anon_key 是 publishable key，可安全暴露在客户端代码中。

## 功能清单

| 模块 | 功能 |
|---|---|
| 认证 | 邮箱密码登录 / 登出 / 路由守卫 |
| 仪表盘 | 图书总数、读者总数、借出中、已逾期实时统计 |
| 图书管理 | 列表（搜索 + 分类筛选 + 分页）、新增、编辑、删除 |
| 读者管理 | 列表（搜索 + 分页）、新增、编辑、删除 |
| 借阅管理 | 列表（状态筛选）、新建借阅、还书、删除；逾期自动标记 |
| 分类管理 | 内联弹窗新增、编辑、删除 |

## 平台支持

| 平台 | 适配 |
|---|---|
| 桌面端（≥ 768px） | 侧栏固定展开，完整表格视图 |
| H5 移动端（< 768px） | 侧栏浮层收起，☰ 汉堡菜单，表格横向滑动，表单单列 |
