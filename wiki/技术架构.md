# 技术架构

## 技术栈

| 层 | 技术 |
|---|---|
| 框架 | Vue 3（Composition API + `<script setup>`） |
| UI 库 | Naive UI 2.x |
| 构建 | Vite 8.x |
| 路由 | Vue Router 4 |
| 状态管理 | Pinia 3 |
| 后端 | Supabase（PostgreSQL + Auth + REST API） |

## 响应式设计

支持桌面端和 H5 移动端：

| 断点 | 侧栏 | 表格 | 表单 |
|---|---|---|---|
| ≥ 768px | 220px 固定展开 | 完整列 | 两列网格 |
| < 768px | transform 浮层 + ☰ 汉堡菜单 | 横向滑动 | 单列全宽 |

侧栏在 AppLayout 中通过 `window.innerWidth` 监听自动切换，`collapse-mode="transform"` 保证移动端不挤压内容区。

## 架构图

```
浏览器
  │
  ├─ Vue 3 (Composition API + <script setup>)
  │   ├─ Naive UI ── 全局 UI 组件
  │   ├─ Vue Router ── 客户端路由 + beforeEach 守卫
  │   └─ Pinia ── 4 个 Store
  │
  └─ @supabase/supabase-js ── REST API
         │
         ▼
Supabase 后端
  ├─ PostgreSQL ── 数据表 + RPC（原子事务）
  ├─ GoTrue ── 认证（Email + Password）
  └─ PostgREST ── 自动 REST API
```

## 数据流

```
View（组件）
  │ 用户操作
  ▼
Store（Pinia）
  │ supabase.from().select/insert/update/delete
  │ supabase.rpc('borrow_book', {...})
  ▼
Supabase REST API
  │ SQL / RPC
  ▼
PostgreSQL
```
