# 数据库设计

## 表结构

### categories（分类）

| 列 | 类型 | 约束 |
|---|---|---|
| id | BIGSERIAL | PRIMARY KEY |
| name | TEXT | NOT NULL, UNIQUE |
| description | TEXT | |
| created_at | TIMESTAMPTZ | DEFAULT now() |

### books（图书）

| 列 | 类型 | 约束 |
|---|---|---|
| id | BIGSERIAL | PRIMARY KEY |
| title | TEXT | NOT NULL |
| author | TEXT | NOT NULL |
| isbn | TEXT | UNIQUE |
| category_id | BIGINT | FK → categories, ON DELETE SET NULL |
| publisher | TEXT | |
| publish_date | DATE | |
| stock | INT | DEFAULT 1, CHECK (>= 0) |
| available | INT | DEFAULT 1, CHECK (>= 0) |
| cover_image | TEXT | |
| description | TEXT | |
| created_at | TIMESTAMPTZ | DEFAULT now() |
| updated_at | TIMESTAMPTZ | |

> `available` ≤ `stock` 由业务逻辑保证，数据库层面无 CHECK 约束。

### members（读者）

| 列 | 类型 | 约束 |
|---|---|---|
| id | BIGSERIAL | PRIMARY KEY |
| name | TEXT | NOT NULL |
| phone | TEXT | |
| email | TEXT | |
| address | TEXT | |
| member_type | TEXT | DEFAULT '普通' |
| join_date | DATE | DEFAULT CURRENT_DATE |
| created_at | TIMESTAMPTZ | DEFAULT now() |

> phone / email 无 UNIQUE 约束，允许重复。

### borrowings（借阅）

| 列 | 类型 | 约束 |
|---|---|---|
| id | BIGSERIAL | PRIMARY KEY |
| book_id | BIGINT | NOT NULL, FK → books ON DELETE CASCADE |
| member_id | BIGINT | NOT NULL, FK → members ON DELETE CASCADE |
| borrow_date | DATE | DEFAULT CURRENT_DATE |
| due_date | DATE | NOT NULL |
| return_date | DATE | |
| status | TEXT | DEFAULT 'borrowed', CHECK (IN 'borrowed','returned','overdue') |
| created_at | TIMESTAMPTZ | DEFAULT now() |

## RPC 函数

所有 RPC 使用 `SECURITY DEFINER`，在数据库所有者权限下运行，保证原子性。

### borrow_book(p_book_id, p_member_id, p_borrow_date, p_due_date) → BIGINT

```
1. FOR UPDATE 锁定目标图书记录
2. 检查 available > 0，否则 RAISE EXCEPTION
3. UPDATE books SET available = available - 1
4. INSERT INTO borrowings (..., status='borrowed')
5. RETURN borrowing_id
```

### return_book(p_borrowing_id) → VOID

```
1. FOR UPDATE 锁定借阅记录
2. 检查记录存在、status != 'returned'，否则抛异常
3. UPDATE borrowings SET return_date = CURRENT_DATE, status = 'returned'
4. UPDATE books SET available = available + 1
```

### update_book_stock(p_book_id, p_new_stock) → VOID

```
1. COUNT(*) 该书当前借出数（status IN 'borrowed', 'overdue'）
2. new_available = p_new_stock - 借出数
3. 若 new_available < 0 → RAISE EXCEPTION '库存不能小于当前借出数量'
4. UPDATE books SET stock = p_new_stock, available = new_available
```

## RLS 策略

所有表启用 RLS，策略统一为：

```sql
FOR SELECT / INSERT / UPDATE / DELETE
TO authenticated
USING (true) / WITH CHECK (true)
```

> 单管理员模式：认证通过即可全量操作所有表。

## 级联删除行为

| 操作 | 影响 |
|---|---|
| 删除图书 | 该图书的所有借阅记录自动删除 |
| 删除读者 | 该读者的所有借阅记录自动删除 |
| 删除分类 | 相关图书的 category_id 设为 NULL |
