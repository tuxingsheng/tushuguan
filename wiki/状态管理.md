# 状态管理

四个 Pinia Store，全部使用 Composition API（`defineStore` + `setup` 函数）。

## auth.js — 认证

| 状态 | 类型 | 说明 |
|---|---|---|
| user | Object | 当前登录用户 |
| loading | Boolean | 登录请求中 |

| 方法 | 说明 |
|---|---|
| `register(email, password)` | 调用 `signUp`，返回 user/session。若邮箱已注册，identities 为空 |
| `login(email, password)` | 调用 `signInWithPassword`，成功设置 user |
| `logout()` | 调用 `signOut`，清空 user |
| `checkSession()` | 调用 `getSession` 恢复会话 |

## books.js — 图书

| 状态 | 类型 | 说明 |
|---|---|---|
| books | Array | 当前页图书列表 |
| total | Number | 符合条件的总记录数 |
| loading | Boolean | 加载中 |

| 方法 | 说明 |
|---|---|
| `fetchBooks({ page, pageSize, search, categoryId })` | 分页 + 搜索 + 分类筛选，联查 `categories(name)` |
| `fetchBook(id)` | 获取单本图书详情 |
| `createBook(book)` | 新增，`available = stock` |
| `updateBook(id, book)` | 更新非库存字段 |
| `updateBookStock(id, newStock)` | **RPC**：安全更新库存（见 database.md） |
| `deleteBook(id)` | 删除（CASCADE 关联借阅记录） |

## members.js — 读者

| 状态 | 类型 | 说明 |
|---|---|---|
| members | Array | 当前页读者列表 |
| total | Number | 总记录数 |
| loading | Boolean | 加载中 |

| 方法 | 说明 |
|---|---|
| `fetchMembers({ page, pageSize, search })` | 分页 + 搜索（name/phone/email） |
| `fetchMember(id)` | 获取单个读者 |
| `createMember(member)` | 新增 |
| `updateMember(id, member)` | 更新 |
| `deleteMember(id)` | 删除（CASCADE 关联借阅记录） |

## borrowings.js — 借阅

| 状态 | 类型 | 说明 |
|---|---|---|
| borrowings | Array | 当前页借阅记录 |
| total | Number | 总记录数 |
| loading | Boolean | 加载中 |

| 方法 | 说明 |
|---|---|
| `fetchBorrowings({ page, pageSize, search, status })` | **前置**：先 `UPDATE status='overdue' WHERE due_date < today AND status='borrowed'`，再查询。联查 `books(title,author,isbn)` 和 `members(name,phone)` |
| `createBorrowing({ book_id, member_id, borrow_date, due_date })` | **RPC `borrow_book`**：原子借书 |
| `returnBook(borrowingId)` | **RPC `return_book`**：原子还书 |
| `deleteBorrowing(id)` | 删除单条记录 |
