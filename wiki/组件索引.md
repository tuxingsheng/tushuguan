# 组件索引

## 布局组件

### AppLayout.vue

主布局框架，支持桌面端和 H5 移动端自适应。

**结构：**
```
n-layout (height: 100vh)
├── n-layout-header (52px)
│   ├── [移动端] ☰ 汉堡按钮 + logo + 标题
│   └── 管理员头像 + 退出下拉菜单
└── n-layout (height: calc(100vh - 52px))
    ├── n-layout-sider (220px / 移动端浮层)
    │   └── SideMenu
    ├── [移动端] 半透明遮罩层（点击关闭侧栏）
    └── n-layout-content
        └── <router-view />
```

**关键逻辑：**
- `activeKey` 通过 `route.path` 前缀匹配计算
- `isMobile`：`window.innerWidth < 768` + `resize` 事件实时监听
- 移动端侧栏默认收起（`collapse-mode="transform"` 浮层模式），☰ 按钮切换
- 桌面端侧栏始终展开
- 移动端点击菜单项自动收起侧栏
- 退出菜单 `n-dropdown` → `handleSelect` → `auth.logout()` + `router.push('/login')`

### SideMenu.vue

封装 Naive UI `n-menu`。5 个固定菜单项：

| Key | 标签 |
|---|---|
| dashboard | 仪表盘 |
| books | 图书管理 |
| members | 读者管理 |
| borrowings | 借阅管理 |
| categories | 分类管理 |

样式覆写在 `<style scoped>` 的 `:deep(.n-menu)` 中。主题色变量来自 `global.css`。

## 视图组件

### Login.vue

登录页面，独立于 AppLayout。

- 深色渐变背景
- 白色卡片居中
- 邮箱 + 密码表单 + Naive UI 验证
- 回车触发登录
- `n-message` 显示成功/失败

### Dashboard.vue

仪表盘统计页。

**三种状态：**

| 状态 | 显示 |
|---|---|
| 加载中 | `n-spin` |
| 加载失败 | `n-result` 500 + 错误描述 + "重新加载"按钮 |
| 数据为空 | 引导卡片：4 步操作指南 |
| 正常 | 4 个统计卡片 + 快速操作卡片 |

**统计卡片：** `cards` 为 `computed`，`stats` 更新时自动刷新。逾期 > 0 时卡片变红。

**数据源：** 4 个并行的 `supabase.from().select('*', { count: 'exact', head: true })` 查询。

### Books.vue

图书列表页。

- 工具栏：搜索框（书名/作者/ISBN）+ 分类下拉 + 搜索按钮
- 表格列：书名、作者、ISBN、分类、出版社、库存(available/stock)
- 操作列：编辑（跳转 BookForm）、删除（Popconfirm，提醒级联影响）
- 空状态：`<n-empty description="暂无图书" />`
- 分页：远程分页，`onChange` 重新加载

### BookForm.vue

图书新增/编辑表单（复用组件）。

- 通过 `route.params.id` 判断新增还是编辑
- 两列网格布局（书名/作者/ISBN/分类/出版社/出版日期/库存/封面URL）+ 简介 textarea
- **编辑模式提交逻辑：**
  1. 分离 `stock` 字段
  2. `updateBook(id, rest)` — 更新非库存字段
  3. `updateBookStock(id, stock)` — RPC 安全更新库存
- 表单验证：书名必填、作者必填、库存 ≥ 0

### Members.vue

读者列表页。

- 工具栏：搜索框（姓名/电话/邮箱）+ 搜索按钮
- 表格列：姓名、电话、邮箱、地址、类型、注册日期
- 操作：编辑/删除（Popconfirm，提醒级联影响）

### MemberForm.vue

读者新增/编辑表单。

- 两列网格：姓名/电话/邮箱/读者类型/注册日期
- 地址：textarea
- 读者类型：普通 / VIP / 学生 / 教师

### Borrowings.vue

借阅记录列表页。

- 工具栏：搜索 + 状态下拉筛选（全部/借出中/已归还/已逾期）
- 表格列：图书（联查名）、读者（联查名）、借阅日期、应还日期、归还日期、状态 Tag
- 状态 Tag 颜色：borrowed=warning、returned=success、overdue=error
- 操作列：
  - 借出中/逾期 → "还书"按钮（Popconfirm）→ RPC `return_book`
  - 全部 → "删除"按钮（Popconfirm）

### BorrowForm.vue

新建借阅表单。

- 图书下拉：只显示 `available > 0` 的图书（显示格式：`书名 (可借: N)`）
- 读者下拉：全部读者
- 借阅日期 + 应还日期选择器
- 提交 → `borrowingsStore.createBorrowing()` → RPC `borrow_book`

### Categories.vue

分类管理页。

- 表格：ID + 名称 + 描述，操作列（编辑/删除）
- 新增/编辑弹窗：`n-modal` preset="card"
- 表单验证：名称必填
- 删除确认：Popconfirm
